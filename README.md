# Mage Versus Mage

Você e seu companheiro mago, aliado de batalha, após passarem por uma missão arduosa em uma masmorra, decidem descansar em uma taverna, sendo os últimos restantes ali, preste a saírem, decidem rever o espólio, tanto para pagar o taverneiro, quanto para dividirem entre si. PORÉM! O dinheiro fora roubado! Furiosos e alternados decidem batalhar para descobrir o que aconteceu de verdade. MATE O OUTRO MAGO! DESCUBRA O QUE ACONTECEU COM O ESPÓLIO!

"Tenho que destruir aquele que jurou me defender...!"

"Não posso perder essa recompensa. Farei-o REFLETIR sobre isso! EM OUTRA VIDA!"

🧙‍♂️ Um jogo onde você é um mago tendo que matar outro do mesmo nível de poder que o seu. TOME CUIDADO!
